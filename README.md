# To-Do List Application

## Description
A responsive To-Do List application built using HTML, CSS, and vanilla JavaScript.  
It allows users to add, edit, delete, and mark tasks as completed. Tasks are saved using Local Storage.

## Features
- Add, edit, delete tasks
- Mark tasks as completed
- Prevent empty task submission
- Persistent data using Local Storage
- Responsive design for mobile and desktop

## How to Run
1. Download or clone the project.
2. Open `index.html` in any modern web browser.
3. Start managing your tasks.

No additional setup required.
